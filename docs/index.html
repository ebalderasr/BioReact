BioReact

Plataforma educativa y de simulación para estudiar la dinámica, estabilidad y control de biorreactores continuos. El proyecto está pensado para avanzar desde balances de masa y cinética Monod hacia casos más complejos como Haldane, inhibición por producto y estrategias de control.

Qué enseña

Cómo pasar de balances de biomasa y sustrato a una interpretación dinámica del reactor, incluyendo washout, estados estacionarios y estabilidad.

Qué simula

Trayectorias temporales, comparación de modelos cinéticos y, más adelante, la respuesta del sistema ante diferentes estrategias de control.

Cómo crecerá

Release 1 deja lista la arquitectura base. Después iremos agregando Monod, Haldane, inhibición por producto, paneles y gráficas.
