Sobre BioReact

BioReact es una plataforma educativa y de simulación enfocada en biorreactores continuos. El proyecto integra una vertiente didáctica y una vertiente de exploración dinámica para conectar balances, cinética, estabilidad y control.

Repositorio

La web vive en /docs y el material de apoyo permanece organizado fuera del sitio.

Notebook

El notebook será la referencia para el análisis simbólico y el motor matemático conforme lo vayamos depurando.

Excel

El libro de Excel conserva el valor didáctico del cálculo paso a paso y servirá como soporte pedagógico.
